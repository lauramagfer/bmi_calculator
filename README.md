# java_bmi_calculator

O objetivo desse projeto é fortalecer a lógica de programação através de uma calculadora de IMC. 

---

### Funcionamento

O programa pode ser utilizado por meio do terminal de uma IDE.<br>
Assim que ele é iniciado, é solicitado ao usuário duas informações: sua altura e seu peso.<br> 
Tendo essas informações, o algoritmo realiza o cálculo de IMC do usuário, trazendo também sua classificação.<br>

### Conceitos aprendidos

Mediante esse projeto, pude aprender mais sobre estruturas condicionais aninhadas, estruturas repetivivas (do-while para o usuário decidir se vai realizar outro cálculo antes de finalizar) e fórmulas.
